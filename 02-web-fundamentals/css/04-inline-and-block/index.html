Display Inline, Inline-Block, Block

The display property defines how an element is rendered on the page.

  • Inline
  • Inline-block
  • Block

Inline

Inline elements do not start on a new line. They flow along with the content. Width and height have no effect on inline elements.

Examples:

  • span
  • a
  • strong
  • em
This goes nowhere.

Block

Block elements always start on a new line and take up the full width available.

Examples:

  • div
  • headings (h1, etc.)
  • p
  • ul, ol

Inline-Block

Inline-block elements combine features of both inline and block elements.

Examples:

  • img
  • input
  • select
  • textarea
