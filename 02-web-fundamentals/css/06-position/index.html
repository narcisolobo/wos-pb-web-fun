The CSS Position Property

The first rule of the position property:
Do not abuse the position property.

When first learning about the possibilities of the position property, it can be tempting to use it all the time. The position property allows us to place any element anywhere we like.

Although this may seem like the answer to our prayers, we should avoid using position whenever possible. We should instead opt for cleaner, more maintainable code. Flex your containers instead!

There are five possible values for the position property. These are (in no particular order):

  • position: static
  • position: relative
  • position: absolute
  • position: fixed
  • position: sticky

Position: Static
This value is the default value for the position property.

Position: Relative
This value positions the element relative to its normal position in the document flow.

Position: Absolute
This value positions the element relative to its closest positioned ancestor.

An absolutely positioned element must have a relatively positioned parent.

Position: fixed
The element is positioned relative to the browser window.

Position: Sticky
The element toggles between fixed and relative, dependant on the client's scroll position.
